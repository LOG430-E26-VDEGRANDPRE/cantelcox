# cantelcox

Référentiel conteneur principal de CanTelcoX.

Fournit :

- Prometheus
- Grafana
- KrakenD
- MySQL
- Mongo
- Postgres
- Redis
- Jaeger

## Dépendances

Tous les services doivent être présent dans l'arborescence (les dossiers svc-* sont dans .gitignore) afin que le docker-compose de ce référentiel puisse les inclure.
